My First AWS EC2 Web Server 

This is my first cloud deployment project using AWS EC2, Ubuntu Linux, and Apache Web Server.

## Project Overview
I launched an EC2 instance on AWS, connected to it using SSH, installed Apache2, configured security groups, and hosted a public webpage.

## Technologies Used
- AWS EC2
- Ubuntu Linux
- Apache2
- HTML
- GitHub

## What I Learned
- Launching EC2 instances
- Connecting to Linux servers using SSH
- Configuring Security Groups
- Allowing HTTP traffic on Port 80
- Hosting webpages publicly
- Editing files using Linux terminal
- Using GitHub repositories and commits

## Webpage Content
```html
<h1>Mahaboob Cloud Server </h1>
<p>I deployed this on AWS EC2</p>
```

## Architecture Flow
Browser → Public IP → Security Group → Apache Web Server → index.html
